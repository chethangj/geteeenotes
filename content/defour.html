CHAPTER 04 : COMBINATIONAL LOGIC CIRCUITS

Adders:

It is combinational circuit which performs the additional binary bits.

There are two types.

  1. Half adder
  2. Full adder

1. Half adder:

It is a logical circuit which performs the operation of addition of two binary bits is called half adder.

It has two inputs and produces output has sum and carry denoted by S and C respectively.

2. Full Adder:

It is logic circuit which performs the operation of addition of 3 binary bits is called full adder.

It has three inputs and produces output has sum and carry denoted by S and C respectively.

Multiplexer:

It is a combinational circuit or a data selector which transmits large number of information over a single line.

4:1 Multiplexer:

Logic Diagram:

Truth Table:

  • It has four input lines and two data select lines because of with each of the four data inputs can be selected.
  • Two bit code on data select line will allow the data on the selected data inout to pass through the data output.
  • If S0=0, S1=0 then the input data will appear on the input line D0. Hence the output will be given by,

    • \[Y = \overline {{S_O}} \overline {{S_1}} {D_0}\]

  • If S0=0, S1=1 then the input data will appear on the input line D1. Hence the output will be given by,

    • \[Y = \overline {{S_O}} {S_1}{D_1}\]

  • If S0=1, S1=0 then the input data will appear on the input line D2. Hence the output will be given by,

    • \[Y = {S_O}\overline {{S_1}} {D_2}\]

  • If S0=1, S1=1 then the input data will appear on the input line D3. Hence the output will be given by,

    • \[Y = {S_O}{S_1}{D_3}\]

  • The final expression for the data output is given by,
    • \[Y = \overline {{S_O}} \overline {{S_1}} {D_0} + \overline {{S_O}} {S_1}{D_1} + {S_O}\overline {{S_1}} {D_2} + {S_O}{S_1}{D_3}\]

Applications:

  • Data selection.
  • Data routing.
  • Operation sequencing.
  • Parallel series connection.
  • Waveform generation.
  • Logic function generation.

De-Multiplexer:

De-Mux is a logic circuit which has one input and many outputs. By using select line we can connect the input signal to one of the output lines.

1:4 De-Multiplexer:

Logic Diagram:

Truth Table:

  • The input data bit is taken as D. the two select lines enables only one output line at a time.
  • The data in the input line D will pass through the enabled gates at the output line.
  • When S0=0, S1=0 then the input data D will be transmitted through first output line. Hence the output will be given by,

    • \[{Y_0} = \overline {{S_O}} \overline {{S_1}} D\]

  • When S0=0, S1=1 then the input data D will be transmitted through second output line. Hence the output will be given by,

    • \[{Y_1} = \overline {{S_O}} {S_1}D\]

  • When S0=1, S1=0 then the input data D will be transmitted through third output line. Hence the output will be given by,

    • \[{Y_2} = {S_O}\overline {{S_1}} D\]

  • When S0=1, S1=1 then the input data D will be transmitted through fourth output line. Hence the output will be given by,

    • \[{Y_3} = {S_O}{S_1}D\]

Applications:

  • Data distribution.
  • Security monitoring system.
  • Synchronous data transmission system.

Applications of Encoder:

  • Alpha numeric to asci code conversion.
  • Alpha numeric to teletype keyboards.

Priority Encoder:

In priority encoder more than one input is set at a time. During such condition the input with highest value is encoded and ignores the other input.

Example: If two and eight set at the same time then the preference is given for eight to encode and 2 will be ignored hence the name priority encoder.

Seven Segment Display:

LED display consisting of 7 LED’s arranged by seven segments as shown in figure is called seven segment displays.

There are two types.

  1. Common anode type
  2. Common cathode type

Working:

To display 0 the LED’s a,b,c,d,e,f are forward biased. Similarly for the number 5 the segments a,f,g,c,d are forward biased hence seven segment display depends on the digits to be display. At that time particular set of LED’s are forward biased.
