CHAPTER 02 : Transistors and MOSFET's

Transistor:

Transistor is three terminal device that is base emitter and collector that can be operated in three configurations common base, common emitter and common collector. This generally used for current or voltage amplification.

Types are,

  1. Unipolar Junction Transistor
  2. Bipolar Junction Transistor

1) In unipolar junction transistor current conduction is only due to the one type of charge carriers that is majority carriers.

2) The current conduction in bipolar transistor because of both type of charge carriers that is electrons and holes hence this is called bipolar junction transistor.

Types of bipolar junction transistor:

  1. NPN
  2. PNP

Formation of NPN and PNP Transistor:

A transistor is formed by sandwiching single P region between two N region this is called NPN transistor.

A transistor is formed by sandwiching single N region between two P region this is called PNP transistor.

The middle region of earth transistor type is called base of the transistor. It is lightly doped the remaining two region are called emitter and collector. The emitter region is heavily doped and collector region is moderately doped.

Symbols and Terminals:

Junction in Transistor:

  • Transistor has two PN junctions.
  • One junction between emitter and base and is called emitter junction base.
  • The outer junction between base and collector is called collector junction.

Unbiased Transistor:

  • Unbiased transistor is a transistor with no external voltage is applied and there will be no current flowing from any treads other transistor.
  • Since transistor is like two PN junction diode connected back to back. There are depletion region at both the junction emitter junction and collector junction a s shown in the figure.
  • During diffusion process, depletion region penetrates more deeply into lightly doped side in order to include an equal number of impurities added in each side of the junction.
  • Depletion region and emitter junction penetrate less in the heavily doped emitter and extends more in base region.
  • Similarly width of the depletion layer of the collector region is less than heavily doped region.

The Working Principle of NPN:

  • Base to emitter region is forward biased DC voltage source VEE thus the depletion region is small the collector to base region is reverse bias increasing depletion region at collector to base junction.
  • Forward bias emitter base junction causes the elections in N type emitter to flow towards base. This constitutes IE. As these electrons flow through the P type base. They tend to combine with holes in P region.
  • Due to very light doping very few of the electrons inject into the base.
  • From the emitter recombine with holes to constitute base current IB.
  • Remaining large number of electrons cross the base region and move through the collector region.

Electrons to flow across emitter to base junction to the positive terminal of the DC source.

  • This constitute collector current IC. Thus the electrons flow constitutes dominant current in the NPN transistor.
  • Since the most of the electrons from emitter flow in the collector in very few combines with holes in the base thus common collector current is larger than the base current.

Working Principle of PNP Transistor:

  • Base to emitter region is forward biased DC voltage source VEE thus the depletion region is small the collector to base region is reverse bias to increasing depletion region at collector to base junction.
  • Forward emitter base junction causes the holes in P type emitter to flow forward base. This constitutes IE. As these holes flow through the N type base they tend to combine with electrons in N region.
  • Due to very light doping very few of the holes inject into base.
  • From the emitter recombine with electrons to constitutes base current IB.
  • Remaining large number of holes cross the base region and move through the collector region holes to flow across emitter to base junction to the negative terminal of the DC source.
  • This constitute collector current IC. Thus the holes flow dominant current in the PNP transistor.
  • Since the most of the holes from emitter flow in the collector in very few combine with electrons in the base thus common collector current is large than the base current.

Transistor Configuration:

  1. Common Base Configuration
  2. Common Emitter Configuration
  3. Common Collector Configuration

Common Base Configuration:

In this configuration input is applied between emitter and base and output is taken from collector to base. Here base of the transistor is common to both input and output circuits and hence the name common base configuration.

Common Emitter Configuration:

In this configuration input is applied between base and emitter and output is taken from collector to emitter. Here emitter of the transistor is common to both input and output circuits and hence the name common emitter configuration.

Common Collector Configuration:

In this configuration input is applied between base and collector and output is taken from collector to emitter. Here collector of the transistor is common to both input and output circuits and hence the name common collector configuration.

Different Regions of Transistor:

The output characteristics have three regions namely,

  1. Active region
  2. Saturation region
  3. Cutoff region
  • Saturation region is shown by a line OA and it is called saturation line. This region both junctions are forward bias and increases in a base current doesn’t cause a corresponding large change in IC.
  • The ration of VCE to IC in this region is called saturation resistance.
  • The region below the curve IB=o called cutoff region and this both the junctions are reverse bias. In this region the operating point enters the cutoff region the transistor said to be OFF. Hence the collector becomes almost zero and the VCE=VCC.
  • The central region the curves are uniform spacing and slope is called active region. In this region the emitter base junction is forward bias where the collector emitter junction is reverse bias. If a transistor should be used as a linear amplifier it should be operated in active region.

Ratings:

Power Dissipation:

  • A transistor conducts current flows between collector and emitter.
  • It drops voltage between those two points at any given time. The power dissipation by a transistor is equal to the product of collector current and collector emitter voltage.
  • Just like resistors, transistors are rated of how many watts each can safety dissipate without sustaining damage. This rating is called maximum allowable power dissipation.

Reverse Breakdown Voltages:

Transistors are rated for maximum allowable reverse breakdown voltage across their PN junction. This includes voltage rating for the emitter base junction VEBO with collector open, collector base junction VCBO with emitter open and also from collector to emitter VCEO with base open.

Collector Current:

ICmax is maximum allowable collector current rating this means that transistor can handle up to ICmax of direct current provided that the power rating is not exceeded.

Transistor as a Switch:

  • The transistor operates in regions namely active region, saturation region and cutoff region.
  • When transistor is made to operate in active region then it acts as an amplifier.
  • When it is made to operate in cutoff region and saturation region it acts as a switch. For power electronic applications transistor is mainly used as a switch. The saturation state occurs when both EB and CB junctions are forward biased. The cutoff state occurs when both the junctions are reversing biased. A transistor can be used as a electronic switch which can turn ON and turn OFF the current.
  • When input base voltage (VBE) is positive the transistor saturates the collector current IC will flows through the collector resistance RC. This condition similar to a closed switch (ON) and the LED will glow.
  • When input base voltage is negative the transistor operates in cutoff region and no collector current (IC) flows through the collector resistance RC. This condition is similar to that of open switch (OFF) and the LED does not glow.

Thermal Run Away:

  • When there is an increase in temperature of transistor this increases the collector current which increases the current gain and hence ICO (leakage current) also increases.
  • In particular ICO changes greatly it doubles for every 100C raises in temperature.
  • IC causes CB junction temperature to rise which in term increases ICO as a result IC will increase still further it will further rise the temperature at the CB junction.
  • The process will become leading to thermal run away.

Q-Point:

  • In order to produce free electrons output in amplifier circuit supply voltages and resistances in the circuit must be suitably choose.
  • These voltages and resistances establish a set of DC voltages (VCEQ – ICQ) to operate the transistor in the active region.
  • The process of giving proper supply voltage and resistance is called Q point.

Heat Sink:

Heat sink is a mechanical device which is connected to the case of the transistor that provides a large surface area to dissipate developed heat. The heat sink carries the heat to the surrounding.

FET (Field Effect Transistor):

  • Field effect transistor is another semiconductor device like a BJT which can be used as a amplifier or switch.
  • Like BJT, FET is also has a three terminals.
  • The three terminals of FET are named as, Drain, Source and Gate.

Unipolar Device:

We know that in BJT current is carried by both electrons and holes hence the name BJT. However in FET current is carried by only one type of charge carriers either electrons or holes. Hence FET is called unipolar device.

Construction of N type Enhancement MOSFET:

  • The silicon oxide layer is present to isolate the gate metallic platform from the region between drain and source.

E-MOSFET:

Operation of E-MOSFET

Working

  • An applying of drain to source voltage VDS and keeping gate to source voltage zero. By directly connecting gate terminal to the source terminal. Practically zero current flows (i.e. ID=0)
  • If we increase the value of VGS there is a measurable current flow between drain and source. This value of VGS is called threshold voltage. Thus we can say that in an enhancement type N channel MOSFET a positive gate voltage above the threshold value induces a channel and hence drain current by creating a thin layer of negative charges in the substrate region adjacent to the SiO2 layer.
  • The conductivity of the channel is enhanced by increase in the gate to source voltage. Thus pulling more electrons to channel.
  • For any voltage bellow than the threshold voltage there is no channel hence no drain current (i.e. ID=0).
  • Since the channel does not exits VGS=0 volts and enhanced by the applying of positive gate voltage. This type of MOSFET is called N type enhancement MOSFET.

Transfer Characteristics:

  • For an N channel MOSFET it is positive voltage applied to VGS when we apply gate voltage zero. There will be no drain current.
  • When we applied positive voltage less than the Vt voltage, still the channel is not formed. So ID=0.
  • When VGS is equal to threshold voltage (VGS=Vt), there is a formation of small channel consisting of negative charged ions which helps in conduction. Hence a very small current start flowing from source t drain.
  • When VGS is greater than Vt (VGS>Vt), there is a formation of wider channel (increase in the width of channel), electrons present as the minority charge carriers start accumulating in P substrate start moving to the region between the source and drain.
  • Hence there is a large flow of current (current starts increasing rapidly).

Drain Characteristics:

When VGS increases behind Vt the density of electrons in an induced channel increases, increases in the drain current. However at the some point of VDS, for constant VGS that drain current reaches a saturation level.

MOSFET as a Switch:

Like transistor MOSFET can also be used as a switch for E-MMOSFET no channel exits while gate is at the same potential as a source. Therefore MOSFET requires no external voltage to switch OFF that is they can be operated, single polarity supply. For the N channel MOSFET positive input voltage greater than the threshold voltage is necessary to switch ON.
